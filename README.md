# tk-theme

## Shortcodes
### Synthwave
`[synthwave]`

### Carousel
`[tk_carousel category="post-category-slug"]`

### Buttons
Type can be either "icon" or "text" (Icon being Font Awesome classes - "fas fa-chevron-down" )  
`[tk_button text="fas fa-chevron-down" href="example.com" type="icon"]`
